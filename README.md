# Memory Parking

## Getting Started

### Prerequisites
- Node.js
- Yarn
- Expo Go (on device) or Emulator (Android Studio/Xcode)

### Installation
\`\`\`bash
yarn install
\`\`\`

### Running the App
\`\`\`bash
yarn start
\`\`\`
- Press \`a\` for Android emulator.
- Press \`i\` for iOS simulator.
- Scan QR code for physical device.

### Testing
\`\`\`bash
yarn test
\`\`\`
